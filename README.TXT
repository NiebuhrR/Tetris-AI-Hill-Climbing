PROJECT TITLE: Homework 3 - Tetris Brain
PURPOSE OF PROJECT: To create an agent that can play Tetris using heuristics
to differentiate between good and bad boards and learning to tune the weights 
of each heuristic. 
VERSION or DATE: November 13, 2017
AUTHORS: Grace Punzalan and Quyen Ha
USER INSTRUCTIONS: Our brain is named GracePunzalanQuyenHaBrain.java; to run, 
simply compile the java files and load the brain into the game either through 
JTetris or JBrainNoGraphics to see how the agent performs.
